# USB Sicherheitsmonitor 

Ein PowerShell-Skript zum Schutz vor gefährlichen USB-Geräten(Einfach externen Speichern, nicht unbedingt USB)  

## Funktionen
- Überwacht angeschlossene USB-Sticks in Echtzeit  
- Erkennt riskante Dateitypen (`.exe, .bat, .ps1, .js, ...`)  
- Löscht automatisch gefährliche Dateien, wenn Passwort falsch ist  
- Sperrt den Bildschirm bei Bedrohung (Passwortschutz)  
- Passwortgeschützt (Standard: `12345678`, 3 Fehlversuche → Blockierung)  

## Hinweis
- Prototyp, zu Lern- und Demonstrationszwecken  
- Sicherheitseinstellungen sollten für den produktiven Einsatz angepasst werden
- Die Datei ist komplett unbedenklich und man kann sie sorgenlos öffnen
